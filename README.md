# 福建醉美乡村旅游投资有限公司 — 官网

旅游投资类企业官网首页，采用左右分栏布局，呈现「美酒香村」与贴近自然生活的氛围。

## 本地预览

在项目根目录启动静态服务后访问首页，例如：

```bash
cd /Users/gongxifacai/Desktop/Gw
python3 -m http.server 8080
```

浏览器打开：<http://localhost:8080>

也可直接用浏览器打开 `index.html`。

## 结构

| 文件 | 说明 |
|------|------|
| `index.html` | 首页：企业介绍、香村掠影、投资方向 |
| `projects.html` | 项目案例（民宿、酒旅、村落活化等） |
| `news.html` | 新闻动态 |
| `contact.html` | 联系表单与公司信息 |
| `css/style.css` | 全站样式与响应式 |
| `js/main.js` | 导航、联系表单（mailto） |
